Task Management System

A full-featured Task Management System built with Flask and Bootstrap. Users can register, log in, complete tasks, and upload files. Each task is timed, and grades (Ontime, Delay, Overdelay) are calculated and stored in the database.

Key Features:

  • User registration and secure login
  • Complete, and submit tasks with file uploads
  • Automatic timer-based grading for each task
  • User credentials and overall grades stored in Flask/MySQL database
  • Local file preview and storage before submission

Future Enhancements: Adding tasks form another source

Tech Stack: HTML, CSS, JS, Bootstrap,Flask,Mysql
